
# RL-RRT-MPC
This repository implements an RRT*-based planning algorithm, with a Model Predictive Control for planned trajectory tracking.
RL is used to update the algorithm parameters.


## Installation

To install and use this COLAV system, perform the following commands
```
cd rl_rrt_mpc
pip install -e .
cd src/rrt-rust
maturin develop

```
